本医院住院部共两层，每层两个三人间，共可入住十二人。（可扩建）
输入日期格式为：mm/dd/yyyy
注：项目连编时显示的payy找不到，payy是费用计算程序接收的参数,我也不知道为什么总是找不到。
公共变量定义(仅包括int中定义的)：
1. 患者信息表单
formset1 int : 首记录(form1中上一条用到） 末记录（form1中下一条用到） op1(form1 上一条 下一条 text5 中用到)
form2 int : n(随机生成代号和床位中用于产生随机数) nnn(当前住院人数,form3中添加后nnn+1)                      
form4 load : p1(list1中的选中数据）m(添加药品种类即撤销次数)

2.患者查询表单
formset1 int : po(患者代号) sum1(患者药品使用种类） kk(患者费用) kj(患者住院天数） sum1,kk,kj(费用计算程序中按地址传递)
form3 int : m1(已缴纳费用) monny(本次缴纳费用)   m1,monny定义为全局变量可在执行缴纳费用表单中调用

3.管理员表单
form3.page3 int : n1(药房当前记录数以及总记录数) n2(使用过的药品的种数) yp(数组，所有药品使用数量)
 
4.医院运行程序
cMenuName (菜单1的真实名字) padname(菜单栏的真实名字）
